# KSP Cricket Frontend

React frontend for live cricket scores.

## Deploy to Cloudflare Pages

### Build Settings:
- **Build command:** `yarn install && yarn build`
- **Build output:** `build`
- **Node version:** 18

### Environment Variable:
- `REACT_APP_BACKEND_URL` - Your Railway backend URL

## Local Development

```bash
yarn install
yarn start
```

## Features

- Live cricket scores
- Auto-refresh (45 seconds)
- Dark stadium theme
- Telegram integration widget
- Mobile responsive
